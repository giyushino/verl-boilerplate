# verl-boilerplate

## Steps to run:
```
git clone https://github.com/giyushino/verl-boilerplate
cd verl-boilerplate
chmod +x install.sh
./install.sh
chmod +x train.sh
./train.sh

```

## Notes
Check out data.py, change the dataset and run it. Change the grading scheme to what's appropriate, also change train.sh as needed
